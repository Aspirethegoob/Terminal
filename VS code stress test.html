Welcome to the VS testing server

About

It is a simple server that servees static files.

How to use

To use this server, you can run the following command.

            python -m

Then upon running the command, you can access the server at local host {here-->} http://localhost8000

How to stop the server

To stop the server, you can press Crtl + C in the terminal.
